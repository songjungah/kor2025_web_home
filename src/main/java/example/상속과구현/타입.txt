
1. 자바 타입
    1. 기본 타입 종류(8가지)
        boolean
        char
        byte, short, int, long
        float, double

    2. 참조 타입 종류( 8가지 외 모두 )
        Class 타입    : String , ArrayList, Math , Dto , Dao 등등
        Array 타입 : String[] , ArrayList[] 등등
        Interface 타입 : List , Map , mapper 등등
        열거 타입 : ~

2. 이름 정의 (관례)
    자바에서 첫글자를 대문자로 쓰는 경우 : 클래스명 , 인터페이스명 , 상수명 , 열거타입명 등등
    자바에서 첫글자를 소문자로 쓰는 경우 : 변수명 , 메소드명 등등

3. 타입 , 변수 , 변수값 차이 :
    1. 타입 : 변수에 저장된 자료/데이터/리터럴/값 의 종류/형태/형식/타입
    2. 변수 : 자료/데이터/리터럴/값 을 저장 할 수 있는 메모리 공간 , +수정가능
    3. 변수명 : 메모리 공간의 상징적인 이름
    4. 변수값 : 메모리 공간의 저장된 값

즉] int a = 3;   int:타입 , a : 변수명 , 3 : 변수값
